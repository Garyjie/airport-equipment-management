## 📊 数据库表清单

共 10 张表 + 3 个视图：

| 表名 | 中文名 | 记录数（示例） | 说明 |
|------|--------|--------------|------|
| `users` | 用户表 | 2 | 系统用户信息 |
| `stations` | 站点表 | 8 | 值机岛、登机口、自助服务区 |
| `counters` | 柜台表 | 17 | 站点下的柜台/登机口 |
| `device_types` | 设备类型表 | 8 | 设备类型定义和自定义属性 |
| `devices` | 设备表 | 30 | 具体设备信息 |
| `device_change_records` | 设备变更记录表 | 3 | 设备移动和状态变更历史 |
| `paper_change_records` | 换纸记录表 | - | CUSS自助机换纸记录 |
| `consumable_records` | 耗材记录表 | - | 通用耗材更换记录 |
| `audit_logs` | 审计日志表 | - | 所有操作的审计记录 |

**详细表结构请查看：**
- [项目详解.md - 完整数据库表结构](./项目详解.md#二-完整数据库表结构)
- [sql/schema_postgresql.sql](./sql/schema_postgresql.sql)

## 💡 常见问题

### Q: 我是新手，应该从哪开始看？
A: 推荐从 [新手入门详解.md](./新手入门详解.md) 开始，用通俗的语言讲解所有概念。

### Q: 我想快速跑起来，看哪份文档？
A: 看 [QUICK_START.md](./QUICK_START.md)，5分钟就能启动项目。

### Q: 我想了解技术细节，看哪份？
A: 看 [项目详解.md](./项目详解.md)，包含完整的数据库设计、API接口、前后端架构。

### Q: 我要部署到服务器，看哪份？
A: 先看 [DEPLOYMENT_GUIDE.md](./DEPLOYMENT_GUIDE.md)，再看 [PRODUCTION_SETUP.md](./PRODUCTION_SETUP.md)。
